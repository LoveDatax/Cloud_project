# CLOUD DATA PROJECT 

This project demonstrates an end-to-end cloud-based data engineering pipeline built on AWS.  
The objective was to ingest raw transactional data, transform it using AWS Glue, store optimized outputs in Amazon S3, and query the processed data using Amazon Athena.

The project follows a modern lakehouse-style architecture using S3 \+ Glue \+ Athena

**Services Used**;

* **Amazon S3** – Data Lake Storage  
* **AWS Glue (ETL Jobs)** – Data Transformation  
* **AWS Glue Data Catalog** – Metadata & Schema Management  
* **AWS Glue Database** – Grouping of tables  
* **Amazon Athena** – SQL Query Engine
